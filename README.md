# TransitOps — Smart Transport Operations Platform

Digitizes the full transport lifecycle: vehicle & driver management, trip
dispatching, maintenance, fuel/expense tracking, and operational analytics —
with role-based access and automatic status transitions enforcing every
business rule from the brief.

## Monorepo layout
- `server/` — Node + Express + Prisma (SQLite) REST API
- `client/` — React + Vite + Tailwind SPA

See `server/README.md` and `client/README.md` for setup.
